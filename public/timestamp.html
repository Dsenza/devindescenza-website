API Basejump: Timestamp microservice

    1) Pass a string as a parameter to this website.
    2) The website will determine if the string parameter contains a unix time stamp or a natural language formatted date.
    3) The website will return both the unix timestamp and the natural language form if a proper string was passed.
    4) If the string wasn't a date the website returns null for both properties

Example usage:

https://vast-beach-2192.herokuapp.com/api/timestamp/1450137600
https://vast-beach-2192.herokuapp.com/api/timestamp/December%2015,%202015
